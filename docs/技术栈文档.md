## 后端技术栈 (Go)

### 核心框架

| 技术 | 用途 |
|------|------|
| Go 1.22+ | 后端服务开发语言 |
| Gin | HTTP 路由、中间件、请求处理 |
| GORM | 数据库 ORM、模型映射、事务管理 |
| Zap | 结构化日志记录 |
| Viper | 配置文件管理、多环境配置 |

### 缓存系统

| 技术 | 用途 |
|------|------|
| go-redis | Redis 客户端 (会话、分布式锁、限流) |
| SQLite (modernc) | LLM 响应磁盘缓存 (DiskCache) |
| LFU Cache | 内存缓存淘汰策略 |

### 任务队列

| 技术 | 用途 |
|------|------|
| Asynq | 基于 Redis 的异步任务队列 |

### 认证授权

| 技术 | 用途 |
|------|------|
| golang-jwt | JWT Token 生成与验证 |
| OAuth2 | 第三方登录 (Google/GitHub/Microsoft) |

### 实时通信

| 技术 | 用途 |
|------|------|
| gorilla/websocket | WebSocket 实时通知推送 |
| SMTP | 邮件发送服务 |

### 监控与文档
