## 💡 快速链接

### 常用命令
```bash
# 启动开发环境
go run cmd/server/main.go

# 运行数据库迁移
migrate -path db/migrations -database "postgresql://..." up

# 运行测试
go test ./...

# 构建生产版本
go build -o bin/server cmd/server/main.go
```

### 常见问题
- **数据库连接失败**: 检查 `.env` 配置和 PostgreSQL 服务状态
- **端口被占用**: 修改 `config/dev.yaml` 中的端口号
- **依赖下载失败**: 配置 Go 代理 `go env -w GOPROXY=https://goproxy.cn,direct`
